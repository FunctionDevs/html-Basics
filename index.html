Introduction to HTML Coding
Welcome to the WebCode hour-long tutorial on HTML coding. This guide is designed to
introduce you to the basics of HTML (HyperText Markup Language), which is the standard
language used to create web pages. By the end of this tutorial, you'll have the foundational
knowledge to start building your own websites.
What is HTML?
HTML is the backbone of most web pages. It uses a system of tags to structure text, images,
links, and other content on a webpage. HTML is not a programming language; it is a markup
language that denes the structure and presentation of web content.
HTML Document Structure
An HTML document is made up of several key components:
1. DOCTYPE Declaration: This is a declaration at the beginning of an HTML document that
species the HTML version being used.
2. HTML Tags: These are used to create elements. Most tags come in pairs with an opening
and closing tag.
3. Head Section: Contains meta-information about the document, such as the title and links
to stylesheets.
4. Body Section: Contains the content that appears on the page.

Before you start coding, you'll need a text editor and a web browser. Some popular text

Basic HTML Tags
HTML Skeleton
Here's a simple HTML document structure:
<!DOCTYPE html>
<html>
<head>
<title>My First Web Page</title>
</head>
<body>
<h1>Welcome to WebCode!</h1>
<p>This is a paragraph in my rst web page.</p>
</body>
</html>
Key Tags
<html>: The root element of an HTML page.
<head>: Contains meta-information and links to scripts and styles.
<title>: Sets the title of the webpage, visible in the browser tab.
<body>: Contains all the content of the page.
<h1> to <h6>: Headings, <h1> being the largest and <h6> the smallest.
<p>: A paragraph of text.
Adding Links and Images
Links
To create a link, use the <a> tag with the href attribute:
<a href="https://www.webcode.com">Visit WebCode</a>
Images
To add an image, use the <img> tag with src and alt attributes:
<img src="image.jpg" alt="Description of image">
Lists and Tables
Lists
HTML supports ordered (<ol>) and unordered (<ul>) lists:
<ul>
<li>Item 1</li>
<li>Item 2</li>
<li>Item 3</li>
</ul>
<ol>
<li>First</li>
<li>Second</li>
<li>Third</li>
</ol>
Tables
Create tables with the <table>, <tr>, <th>, and <td> tags:
<table>
<tr>
<th>Name</th>
<th>Age</th>
</tr>
<tr>
<td>Alice</td>
<td>30</td>
</tr>
<tr>
<td>Bob</td>
<td>25</td>
</tr>
</table>
Forms
Forms collect user input using various input types:
<form>
<label for="name">Name:</label>
<input type="text" id="name" name="name">
<label for="email">Email:</label>
<input type="email" id="email" name="email">
<input type="submit" value="Submit">
</form>
Conclusion
In this hour-long tutorial, you've learned the basics of HTML, including how to structure a
document, add links and images, create lists and tables, and build simple forms. With this
foundation, you're ready to start exploring more advanced topics like CSS and JavaScript to
enhance your web pages.
Remember, practice is essential. The more you code, the more procient you'll become.
Happy coding, and welcome to the world of web development!
